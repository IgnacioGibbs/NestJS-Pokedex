# Ejecutar el proyecto en desarrollo

1. Clonar el repositorio.
2. Ejecutar.

```bash
npm install
```

3. Tener Nest CLI instalado.

```bash
npm i -g @nestjs/cli
```

4. Levantar la base de datos.

```bash
docker-compose up -d
```

## Stack usado

* MongoDB
* NestJS
